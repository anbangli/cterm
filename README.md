# CTerm
CTerm (Clever Terminal) 不仅可以作为普通 Telnet 客户软件用于 Telnet 站点的登录，更是针对国内BBS的特点设计的一个专用上站软件。 它在运行中对用户和服务器之间的信息进行了分析，知道用户在 BBS 上的当前状态(主选单/讨论区列表/用户列表/文章 列表/编辑状态)，从而提供相应的服务。
